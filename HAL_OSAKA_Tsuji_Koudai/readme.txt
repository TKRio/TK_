//**************************************************************************************
// 作品概要
//**************************************************************************************

・自作したPMDLoderで描画されたモデルを、MeshCutの技術を用いて
  切断が出来る

・Unityを用いてMeshCutを使ったミニゲームを制作した

＊詳しくはプレゼン資料参照

//**************************************************************************************
// 制作期間
//**************************************************************************************
 
・MeshCut     : 3か月ほど
・ミニゲーム  : 1週間

//**************************************************************************************
// 動作環境
//**************************************************************************************

・MeshCut    : DirectX11
・ミニゲーム : Unity 2018.4.10.1f(64bit)

//**************************************************************************************
// 操作説明
//**************************************************************************************

・MeshCut
　　タイトル
　　　Space 		: メインシーンに移行

　　メイン
　　　WASD 		: モデルの移動
　　　矢印 		: カメラ回転
　　　shift＋矢印	: モデルの回転
　　　R			: タイトルに戻る
　　　数字1〜5		: モデルの切り替え　
	モデル
	１：Cube 
	２：Sphere 
	３：Chair
	４：Hart
	５：Character

・ミニゲーム
　　タイトル
　　　Enter		: ゲームシーンに移行
　　　
　　ゲーム
　　　マウス移動	: 刀の移動
　　　左クリックしながらドラッグ＆ドロップ　：　ドロップ時切断実行

　＊ルールは単純。objectが落ちてくるので、それを切断できればscoreが加算されていく。
    切断出来なければゲームオーバー

//**************************************************************************************
// 作品PRポイント
//**************************************************************************************

・PMDローダーとMeshCutを独学で学び、実装した
・以前まで出来ていなかった多頂点モデルを切断できるようにした
・大阪就プレから東京就プレに向けて、一週間弱でミニゲームを制作した
・プレゼン資料やポートフォリオを制作することで、プログラマだけでなく
　他職種の方々のニーズにも答えられるようにした。
